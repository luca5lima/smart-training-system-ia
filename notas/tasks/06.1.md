# List Workout Plan

## Crie uma rota de GET /workout-plans que recebe um filtro no query param "active" opicional. Para cada dia, retorne todos os dados, incluindo exercises.

## O "active" é opcional. Se ele não estiver presente, traga todos os planos de treinos, ativos e inativos.
